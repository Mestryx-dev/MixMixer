# MixMixer v0.1.4-beta.2

**Pre-release / beta** — live settings UX.

## Highlights

- **Live settings**: every change takes effect immediately — no **Apply** / **Cancel**.
- **Listen volume** is hot: headphone monitor level updates while you drag the slider (CABLE / Discord stay at unity).
- Config still auto-saves to `%APPDATA%\MixMixer\config.json` (debounced; flushed when the window hides).

## Install (Windows x64)

1. Download **`MixMixer-v0.1.4-beta.2-windows-x64.zip`** from [Assets](https://github.com/Mestryx-dev/MixMixer/releases/tag/v0.1.4-beta.2).
2. Extract and double-click `mix-mixer.exe` (VB-Cable required).
3. Enable **Headphone monitor** and drag **Listen volume** — level changes in real time.
4. Discord / games mic = **CABLE Output**.

Walkthrough: [docs/TUTORIAL.md](https://github.com/Mestryx-dev/MixMixer/blob/main/docs/TUTORIAL.md)

## Upgrade notes

- Apply / Cancel are gone; leave the window open and tweak freely.
- From beta.1: old `gains.*` keys remain ignored; config rewrite happens on the next auto-save.
