## Risques identifiés

1. **Double APO** (UniteFx + EqualizerAPO) sur le même micro — latence cumulée, ordre de traitement à valider.
2. **Chaîne VST longue** — 2× Limiter6 avec `Latency 1` activé.
3. **VB-Cable isolé** — installé mais sans routage vers le micro virtuel.
4. **Behringer offline** — config `configAPO.txt` orpheline tant que l'interface n'est pas reconnectée.
